# Bitácora 10/03/2026

## Glosario Añadido

Se documentaron los siguientes términos:

## Conceptos Fundamentales

### Feature
Funcionalidad específica que aporta valor al usuario final.

### Feature Scope
Límites y especificaciones detalladas de una característica (qué está incluido y qué queda fuera).

## Gestión de Trabajo

### Backlog
Lista priorizada de trabajo pendiente que debe realizarse en el proyecto.

### Issue
Elemento de trabajo registrado para seguimiento (bugs, tareas, mejoras).

### Weekly
Actividades, reuniones o ciclos que ocurren cada semana (standups, releases, reportes).

## Requisitos

### Requisito Funcional (RF)
Especificación de QUÉ debe hacer el sistema desde la perspectiva del usuario.

### Requisito No Funcional (RNF)
Especificación de CÓMO debe comportarse el sistema (rendimiento, seguridad, calidad).

## Documentación y Calidad

### Bitácora
Registro cronológico de progreso, decisiones e incidentes del proyecto.

### Revisión IA
Análisis automatizado con inteligencia artificial de código, requisitos o seguridad.
