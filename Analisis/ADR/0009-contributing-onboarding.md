# ADR 0009 — Onboarding y `CONTRIBUTING.md` mínimo

Fecha: 2026-05-18

Estado: Propuesta

Contexto
- Nuevos colaboradores deben poder ejecutar scripts (p.ej. métricas) y seguir normas de contribución.

Decisión
- Mantener un `CONTRIBUTING.md` con pasos mínimos: crear venv, instalar dependencias, ejecutar `ScriptMetricas/python metricas.py`, y formato de commits/PRs.

Contenido clave
- Setup local (venv, pip install PyGithub).
- Cómo ejecutar el script de métricas.
- Enlace a ADRs relevantes (0001, 0003, 0006).
